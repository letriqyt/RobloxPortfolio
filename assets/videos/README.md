# Video Clips

Place local portfolio clips in this folder if you want to host them with the site.

Example value in `script.js`:

```js
video: "assets/videos/combat-demo.mp4"
```

You can also use a YouTube URL or video ID:

```js
video: "https://www.youtube.com/watch?v=YOUR_VIDEO_ID"
```

For GitHub Pages, keep video files reasonably small. YouTube is usually better for long clips.
